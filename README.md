# luke-randolph-portfolio

Personal portfolio for Luke Randolph — full-stack web developer.

Built with Next.js 16 (App Router, Turbopack), React 19, and Tailwind CSS v4. Deployed on Vercel.

## Develop

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000.

## Build

```bash
pnpm build
pnpm start
```
